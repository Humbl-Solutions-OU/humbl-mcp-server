# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public GitHub issue.

Email us directly at: security@humbl.ai

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We'll respond within 48 hours and aim to release a fix within 7 days for critical issues.

## API Key Security

Your Humbl.ai API key grants access to your data. Keep it safe:
- Never commit it to version control
- Rotate it immediately if compromised at `https://humbl.ai/admin/mcp/mcpapikey/`
